Technology Stack

Every tool is free, open-source, and production-grade.

WireGuard
VPN tunnel. Punches through NAT. Curve25519 + ChaCha20.
Caddy v2
Reverse proxy. Auto TLS. Wildcard cert via DNS-01.
Docker + Compose
Container orchestration. Isolated bridge network.
Oracle Cloud Free
Always-Free VPS. Stable public IP. Relay only.
DuckDNS
Free wildcard domain. *.caktus.duckdns.org
Let's Encrypt
Free TLS certificates. Auto-renewing. ACME DNS-01.
Uptime Kuma
Self-hosted monitoring. Status page. Alerts.
Ubuntu 22.04
Server OS. UFW firewall. fail2ban. Netplan.
